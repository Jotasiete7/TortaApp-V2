# Registro de Alterações

Todas as mudanças notáveis do TortaApp serão documentadas neste arquivo.

## [Não Lançado]

### Recursos Planejados
- Sistema de feedback no app
- Versão mobile
- Alertas de preço em tempo real
- Recursos sociais (amigos, grupos)

## [0.1.0-beta] - 2025-12-06

### Adicionado
- 🎉 Lançamento beta inicial
- Autenticação de usuário (email/senha)
- Sistema de verificação de nick
- Upload de logs de trade (.txt, .log)
- Busca avançada com operadores
- Dashboard de inteligência de mercado
- Preditor de preços ML
- Sistema de gamificação (XP, níveis, badges)
- Check-in diário
- Leaderboards
- Sistema de badges (12+ conquistas)
- Painel admin
- Upload em massa (NDJSON)
- Sistema de preços de referência
- Insights de preço
- Sistema de mensagens ticker
- UI responsiva com tema escuro

### Recursos

#### Autenticação & Perfil
- Registro por email/senha
- Verificação de email
- Redefinição de senha
- Personalização de perfil

#### Inteligência de Mercado
- Banco de dados de trades
- Busca e filtragem em tempo real
- Ordenação por múltiplos campos
- Comparação de preços
- Filtragem por tipo/raridade

#### Preditor ML
- Previsões para 7 dias
- Intervalos de confiança
- Análise de tendências
- Visualização de dados históricos

#### Gamificação
- 5 níveis de carreira
- 12+ badges desbloqueáveis
- Bônus de XP diário
- Leaderboards públicos

### Técnico
- React + TypeScript
- Vite build system
- Supabase backend
- Row-level security (RLS)
- Lucide React icons

---

## Problemas Conhecidos

- XP atualiza a cada ~5 minutos
- Predições ML requerem dados históricos
- Upload limitado a 50MB
