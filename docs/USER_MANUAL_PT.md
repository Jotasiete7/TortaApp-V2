## 🚀 Primeiros Passos

### O que é o TortaApp?

TortaApp é seu companheiro inteligente para o ecossistema de trading do Wurm Online. Ele analisa dados de trade, prevê preços usando machine learning e gamifica sua experiência de trading com XP, níveis e badges.

### Primeiro Acesso

1. **Abra o TortaApp** - Clique duas vezes no ícone do aplicativo
2. **Crie uma Conta** - Clique em "Sign Up" e insira seu email
3. **Verifique o Email** - Confira sua caixa de entrada para o link de verificação
4. **Login** - Use suas credenciais para acessar o dashboard

## 👤 Configuração da Conta

### Vinculando seu Nick do Jogo

Para rastrear suas estatísticas de trading, você precisa verificar seu personagem do Wurm:

1. Navegue até **Profile** → **Settings**
2. Clique em **"Link Game Nick"**
3. Digite o nome exato do seu personagem Wurm
4. Siga os passos de verificação
5. Uma vez verificado, suas estatísticas começarão a ser rastreadas automaticamente

&gt; **💡 Dica:** O nome do seu personagem é case-sensitive!

## 📤 Upload de Logs de Trade

### Formatos Suportados
- `.txt` - Logs padrão de chat de trade do Wurm
- `.log` - Formato alternativo de log

### Como Fazer Upload
1. Navegue até **Dashboard** → **Upload Logs**
2. Clique em "Choose File" ou arraste e solte
3. Aguarde o processamento
4. Revise os resultados

## 📊 Inteligência de Mercado

### Busca Avançada

```
iron ore ql&gt;90 price&lt;50
```

### Operadores de Busca

| Operador | Exemplo | Descrição |
|----------|---------|-----------|
| `ql&gt;X` | `ql&gt;90` | Qualidade maior que X |
| `price&lt;X` | `price&lt;50` | Preço menor que X copper |
| `qty&gt;X` | `qty&gt;10` | Quantidade maior que X |

## 🎮 Sistema de Gamificação

### Ganhando XP

| Ação | XP |
|------|-----|
| Vender Item (WTS) | +10 XP |
| Comprar Item (WTB) | +10 XP |
| Price Check (PC) | +10 XP |
| Login Diário | +10 XP |

### Níveis

| Nível | Título | Requisito |
|-------|--------|-----------|
| 1 | Novice | 0 - 50 Trades |
| 2 | Apprentice | 50 - 150 Trades |
| 3 | Merchant | 150 - 500 Trades |
| 4 | Veteran | 500 - 1,000 Trades |
| 5 | Tycoon | 1,000+ Trades |

### Badges

Desbloqueie conquistas completando desafios. Veja todas as badges em **Rules &amp; Compendium**.
