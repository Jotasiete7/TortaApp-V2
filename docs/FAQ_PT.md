# Perguntas Frequentes (FAQ)

## Perguntas Gerais

### O que é o TortaApp?
TortaApp é uma ferramenta de inteligência de mercado e análise de trading para Wurm Online. Ajuda você a rastrear preços, prever tendências de mercado e gamificar sua experiência de trading.

### O TortaApp é gratuito?
Sim! O TortaApp é completamente gratuito durante o período beta.

### Meus dados estão seguros?
Sim. Todos os dados são armazenados de forma segura no Supabase com segurança em nível de linha. Nunca compartilhamos suas informações pessoais.

## Conta & Configuração

### Como criar uma conta?
1. Clique em "Sign Up" na tela de login
2. Insira seu email e senha
3. Verifique seu email
4. Faça login e comece a usar o app

### Como verificar meu nick do jogo?
1. Vá para Profile → Settings
2. Clique em "Link Game Nick"
3. Digite o nome do seu personagem Wurm (case-sensitive)
4. Complete o processo de verificação

### Esqueci minha senha. O que fazer?
Clique em "Forgot Password" na tela de login e siga as instruções por email.

## Upload & Dados

### Quais formatos de arquivo são suportados?
- Arquivos `.txt` (logs de chat de trade do Wurm)
- Arquivos `.log`
- **NDJSON** (somente admin, para uploads em massa)

### Como fazer upload de logs de trade?
1. Navegue até Dashboard
2. Clique em "Upload Logs"
3. Selecione seu arquivo ou arraste e solte
4. Aguarde o processamento

## Inteligência de Mercado

### Como buscar itens?
Use a barra de busca no Trade Database:
- **Busca simples:** `iron ore`
- **Busca avançada:** `iron ore ql>90 price<50`

### Quais são os operadores de busca?
- `ql>X` / `ql<X` - Filtros de qualidade
- `price>X` / `price<X` - Filtros de preço
- `qty>X` - Filtros de quantidade

## Preditor de Preços ML

### Quão precisas são as predições?
A precisão depende de dados históricos e volatilidade do mercado. Predições com alta confiança (>80%) são geralmente mais confiáveis.

## Gamificação

### Como ganhar XP?
- Upload de logs de trade (WTS, WTB, PC)
- Check-in diário (+10 XP)
- Cada trade dá +10 XP

### Meu XP não está atualizando. Por quê?
- XP atualiza a cada ~5 minutos
- Tente atualizar a página
- Certifique-se de que seu nick está verificado
